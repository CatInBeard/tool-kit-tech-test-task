Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with test-task API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API use jwt. You need to get token by POST to /api/auth with login and password.
Next you should add header Authorization : Bearer token

Endpoints

Invoke jwt token

Example request:
curl --request DELETE \
    "http://localhost/api/auth" \
    --header "Authorization: Bearer token" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/auth"
);

const headers = {
    "Authorization": "Bearer token",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "DELETE",
    headers,
}).then(response => response.json());

Example response (200):


{
    "message": "User logged out successfully"
}

Example response (401):


{
    "error": "Unauthenticated"
}

You can check the Dev Tools console for debugging information.

    \"name\": \"qui\",

    \"phone\": \"et\",

Update questionary

Available only for admin or to edit yourself

Example request:
curl --request PUT \
    "http://localhost/api/questionary/2" \
    --header "Authorization: Bearer token" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "name=qui"\
    --form "email=muhammad.bartoletti@example.com"\
    --form "password=F`{x6`&P+&r(T*J/beL"\
    --form "phone=odit"\
    --form "tg_name=ut"\
    --form "catPhoto=@/tmp/phpetb4poc6b3da5yk6sjg" \
    --form "cat_photo=@/tmp/phpcgc6iqqjnoqtbsI0H6P" 
const url = new URL(
    "http://localhost/api/questionary/2"
);

const headers = {
    "Authorization": "Bearer token",
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('name', 'qui');
body.append('email', 'muhammad.bartoletti@example.com');
body.append('password', 'F`{x6`&P+&r(T*J/beL');
body.append('phone', 'odit');
body.append('tg_name', 'ut');
body.append('catPhoto', document.querySelector('input[name="catPhoto"]').files[0]);
body.append('cat_photo', document.querySelector('input[name="cat_photo"]').files[0]);

fetch(url, {
    method: "PUT",
    headers,
    body,
}).then(response => response.json());
